FLUXO DE DADOS:
═══════════════
1. User interage com Component
2. Component usa Hook para estado/comunicação
3. Hook chama API Client
4. API Client faz requisição HTTP para Backend
5. Resposta retorna e atualiza estado
6. Component re-renderiza com novos dados

PRINCÍPIOS:
═══════════
✓ Component-based architecture
✓ Server-side rendering (SSR)
✓ Static site generation (SSG)
✓ Client-side interactivity
✓ Type safety com TypeScript
✓ Responsive design
✓ Accessibility (a11y)
